# Discord announcement template

I have released a free community test build of **OSSM HereSphere Controller**.

It connects HereSphere directly to a compatible OSSM through native BLE Streaming, automatically loads matching funscripts from a selected folder, and does not require Intiface or MultiFunPlayer.

Main features:
- automatic HereSphere discovery on the local network;
- direct OSSM BLE Streaming connection;
- repository-based automatic funscript matching;
- manual one-off script loading;
- Reverse and optional Simplified R+D modes;
- pause/stop auto-park;
- optional local diagnostic logs;
- Windows 10/11 x64 installer.
